# 外部データベースへのアクセス（FDA）{#accessing-an-external-database-fda}

Adobe Campaign では、Federated Data Access（FDA）オプションを利用することができます。このオプションを使用すると、1 つ以上の外部データベースに格納されている情報を処理することが可能です。Adobe Campaign データの構造を変更しなくても、外部データにアクセスできます。

以下のような複数のアクティビティで、外部データベースとのデータのやり取りが可能です。

* クエリ
* 分割
* エンリッチメント
* データの読み込み

FDA オプションを使用するには、Adobe Campaign のライセンスが必要です。このオプションの設定やベストプラクティス、利用に関する制限について詳しく説明します [この節](https://experienceleague.adobe.com/docs/campaign/campaign-v8/connect/fda.html?lang=ja){target=&quot;_blank&quot;}。
