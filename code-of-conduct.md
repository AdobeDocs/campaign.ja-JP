# アドビの行動規範

## アドビの誓約

オープンで歓迎される環境を促進するために、我々は
コントリビューターと管理者は、プロジェクトへの参加を約束し、
年齢、身体を問わず、誰もがハラスメントのない体験を提供します
サイズ、障害、民族、性自認と表現、経験レベル，
国籍、容姿、人種、宗教、性自認
方向：

## アドビの標準

ポジティブな環境の創出に貢献する行動の例
次を含める：

* 友好的かつ包括的な言葉を使用する
* 異なる視点や経験を尊重する
* 建設的な批判を素直に受け入れる
* コミュニティにとって最善なことに重点を置く
* 他のコミュニティメンバーに共感を示す

参加者による受け入れがたい行動には、例えば、次のものがあります。

* 性的な言葉や画像の使用と歓迎されない性的注意
アドバンス
* 荒らし、侮辱的／軽蔑的なコメント、個人攻撃または政治的攻撃
* 公的または私的な嫌がらせ
* 物理的または電子的な他人の個人情報を公開する
明示的な許可なしにアドレス
* 合理的に不適切と考えられる他の行為
プロフェッショナルな設定

## アドビの責任

プロジェクトマネージャーは、許容可能な基準を明確にする責任があります
適切かつ公正な是正措置を講じることが期待される
容認できない行動のインスタンスへの対応。

プロジェクト管理者は、プロジェクトを削除、編集、削除する権利と責任を持ちます
コメント、コミット、コード、Wiki編集、イシュー、その他の投稿を却下する
この行動規範に沿っていない、または一時的に禁止する、または
不適切と思われる他の行動の貢献者を永続的に残します，
脅迫、不快、有害な行為：

## 範囲

この行動規範は、プロジェクトスペース内とパブリックスペースの両方に適用されます
個人がプロジェクトまたはコミュニティを代表する場合。 例
プロジェクトまたはコミュニティを表すには、公式のプロジェクト電子メールを使用することが含まれます
住所、公式ソーシャルメディアアカウントを介した投稿、または任命された人物としての行動
オンラインイベントまたはオフラインイベントの代表者。 プロジェクトの表現は
プロジェクトマネージャーによる定義と明確化。

## 実施

虐待、嫌がらせ、またはその他の方法で容認できない行動の事例は次のようなものがあります
プロジェクトチーム（Grp-opensourceoffice@adobe.com）に連絡して報告します。 すべて
苦情は審査および調査され、その結果、以下の対応が行われます
必要かつ適切であると認められる場合。 プロジェクトチームは
事件の報告者に関する秘密を保持する義務があります。
具体的な実施ポリシーの詳細が別途投稿される場合があります。

行動規範を遵守しない、または実施しないプロジェクト管理者
信仰は、他の人によって決定された一時的または恒久的な影響を受ける可能性があります
重要な役割を果たします。
