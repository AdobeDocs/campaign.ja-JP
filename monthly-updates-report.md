# 月次ドキュメント更新レポート

## レポート：2026年2月1日～2026年2月24日

### 新規ページ

_この期間に新しいページはありません。_

### 更新されたページ

- [ キャンペーンセキュリティのベストプラクティス ](help/v8/config/security.md) - 2月19日 – FRR-RSC-05、FRR-RSC-08、FRR-RSC-09。 Managed Cloud Services スコープのメモとセキュリティ チェックリストを上部に追加しました。 新しいセクション：データとPIIの保護（HTTPS、PII制限、パスワード、機密ページ）、認証とセッション（Adobe IMS、セッションのタイムアウト）、インスタンスとネットワークセキュリティ（IP アドレスの許可リストに加える、URL権限、GPG キー）、運用とコンプライアンス（ベースライン比較、監査証跡）。 拡張アクセス管理（セキュリティグループ、管理者制限、製品プロファイル）およびコーディングガイドライン（ハードコーディングされたシークレットなし、入力検証、SQL許可リスト）。 FRR-RSC-08: Campaignは機械読み取り可能なガイダンスを提供していないことを示します。Campaign Settings FAQにリンクして、設定を推奨されるデフォルトと比較します。
- [Campaign Enhanced Security アドオン ](help/v8/config/enhanced-security.md) - 2月19日 – FRR-RSC-09。 「Campaign Enhanced Security アドオンの公開可能なセキュア設定ガイダンス」に設定されています。 このページは、Adobeの一般に公開されている、Campaign v8の推奨されるセキュア設定ガイダンスの一部であり、security.md#public-guidanceへのリンクが含まれていることを示す冒頭の文を追加しました。
- [よくある質問](help/v8/start/campaign-faq-comprehensive.md) - 2月19日 – FRR-RSC-05。 Campaign設定の新しいFAQ:「オペレーターと権限がセキュリティのベストプラクティスに従っていることを確認するにはどうすればよいですか？」 現在の設定の表示場所（管理/アクセス管理、Campaign コントロールパネル）、推奨されるセキュアベースライン（セキュリティガイドライン +強化されたセキュリティアドオン）、および管理者が設定を文書化された推奨事項と比較する方法について説明します。 権限、セキュリティガイドライン、強化されたセキュリティアドオンを使い始めるためのリンク。
- [外部アカウントの設定](help/v8/config/external-accounts.md) - 2月18日 – 外部アカウントの設定と設定が改善されました。 Databricksおよびその他のFDA コネクタドキュメントが更新されました。
- [Federated Data Access （FDA） ](help/v8/connect/fda.md) - 2月6日 – FDAの設定と使用について説明しました。
- [製品ガードレール ](help/v8/start/ac-guardrails.md) - 2月6日 – ガードレールと制限が更新されました。
- [Campaign Classic v7からCampaign v8](help/v8/start/v7-to-v8.md)への移行 – 2月6日 – 移行ガイダンスが改善されました。
- [ イベントの説明を理解](help/v8/send/event-description.md) - 1月27日 – v26.1用に更新されたイベントの説明ドキュメント。
- [時間制約アクティビティ ](help/automation/workflow/time-constraint.md) - 1月27日 – v26.1用に更新された時間制約ワークフローアクティビティのドキュメント。
